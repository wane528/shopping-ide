## 🔧 表格格式修复

### 问题
原来的表格使用了单个破折号，导致渲染失败：
```markdown
| Column 1 | Column 2 |
|----------|----------|  ❌ 单破折号
```

### 解决方案
使用标准 Markdown 格式（3 个破折号）：
```markdown
| Column 1 | Column 2 |
| --- | --- |  ✅ 三破折号
```
